🔄 Part 6: Update & Delete

1. UPDATE

Modifies existing data in a table.

UPDATE students
SET grade = 98.00, age = 21
WHERE id = 1;
⚠️ CRITICAL: Always use a WHERE clause. Without it, EVERY row in your table will be updated!

2. DELETE

Removes entire rows from a table.

DELETE FROM students
WHERE id = 3;
⚠️ CRITICAL: Always use a WHERE clause. Without it, EVERY row in your table will be deleted!

3. Best Practice: Test with SELECT

Before running an UPDATE or DELETE, run a SELECT to see exactly which rows you are about to change:

-- Check first
SELECT * FROM students WHERE age > 21;

-- Then delete
DELETE FROM students WHERE age > 21;
