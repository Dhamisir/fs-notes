🛠️ Part 2: Database Operations

Before creating tables, we need to create and select a "Library" (Database) to work in.

1. CREATE DATABASE

Creates a new empty database container.

CREATE DATABASE school_db;

2. USE DATABASE

Selects which database you want to work with. You MUST do this before creating tables.

USE school_db;

3. DROP DATABASE

Deletes the entire database and everything inside it.

⚠️ Danger: This deletes ALL data permanently and cannot be undone! 
DROP DATABASE school_db;

-- Safe way to drop
DROP DATABASE IF EXISTS school_db;
Typical Workflow:
  1. Create Database
  2. Use Database
  3. Create Tables (Next Topic)
