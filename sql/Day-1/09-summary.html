📝 Course Summary & Exercises

Quick Cheat Sheet

  • CREATE DATABASE db_name; - Create new DB
  • USE db_name; - Select DB
  • CREATE TABLE table_name (...); - Create Table
  • INSERT INTO table_name VALUES (...); - Add Data
  • UPDATE table_name SET ... WHERE ...; - Modify Data
  • DELETE FROM table_name WHERE ...; - Remove Data
  • TRUNCATE TABLE table_name; - WIPE ALL DATA

🎯 Practice Exercises

Exercise 1: Store Management

  1. Create a database named shop_db.
  2. Create a table products with: id (AI/PK), name (VARCHAR), price (DECIMAL), and stock (INT).
  3. Bulk insert 5 products.
  4. Update the price of the first product.
  5. Delete any product with 0 stock.

Exercise 2: Employee Records

  1. Create an employees table.
  2. Add a column for department using ALTER TABLE.
  3. Insert 3 employees.
  4. Try TRUNCATE to see the ID reset to 1.
