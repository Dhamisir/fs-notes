📥 Part 5: Insert Data

1. Basic INSERT INTO

Used to add new rows (records) to a table.

INSERT INTO students (name, age, email, grade)
VALUES ('John Doe', 20, 'john@email.com', 95.50);

2. Multiple Rows in One Query

This is much faster than running individual queries.

INSERT INTO students (name, email)
VALUES 
    ('Sarah Smith', 'sarah@email.com'),
    ('Mike Johnson', 'mike@email.com'),
    ('Emily Brown', 'emily@email.com');
Key Rules for Inserting:
  • Text and Dates must be in single quotes (' ').
  • Numbers do NOT need quotes.
  • Order of values must match the order of columns.
  • You can skip AUTO_INCREMENT columns.
