🗄️ Part 1: Introduction to Databases

A Database is an organized collection of structured information, or data, typically stored electronically in a computer system.

Why do we need Databases?

Imagine managing 1000 students' info in a txt file. It would be slow to search, hard to update, and easy to lose. Databases provide fast searching, security, and the ability to link related data.

1. What is a Database?

Think of a database as a digital library:

  • Database = The Library
  • Tables = Bookshelves
  • Rows = Individual Books
  • Columns = Info about each book (Title, Author)

2. Data vs Information

  • Data: Raw facts (e.g., 25, "John")
  • Information: Data with context (e.g., Age: 25, Name: John)

3. Types of Databases

  1. Relational (SQL): Data in tables. Examples: MySQL, PostgreSQL.
  2. NoSQL: Document or Key-Value based. Examples: MongoDB, Redis.
🚀 Focus: In this course, we will master Relational Databases (SQL) using MySQL.
