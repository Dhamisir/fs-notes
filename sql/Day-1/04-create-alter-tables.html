🏗️ Part 4: Create & Alter Tables

1. CREATE TABLE

Defines the structure of your data table.

CREATE TABLE students (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    age INT,
    email VARCHAR(255) UNIQUE,
    grade DECIMAL(3, 2),
    is_active BOOLEAN DEFAULT TRUE
);

2. Table Constraints

  • PRIMARY KEY: Unique identifier for each row.
  • NOT NULL: Column must have a value.
  • UNIQUE: No duplicates allowed in this column.
  • DEFAULT: Value used if none provided.

3. ALTER TABLE

Changes the structure of an existing table.

Add Column

ALTER TABLE students ADD COLUMN phone VARCHAR(20);

Modify Column

ALTER TABLE students MODIFY COLUMN name VARCHAR(200);

Drop Column

ALTER TABLE students DROP COLUMN phone;

4. DROP TABLE

DROP TABLE IF EXISTS students;
